# 🤖 Mega Tag Bot

Мощный Telegram бот для тегания участников, рассылок и модерации.

## 🚀 Установка

```bash
pip install -r requirements.txt
```

## ⚙️ Настройка

1. Создай бота у [@BotFather](https://t.me/BotFather) → `/newbot`
2. Скопируй токен
3. Открой `bot.py` и вставь токен:
   ```python
   BOT_TOKEN = "1234567890:ABCdef..."
   ```
4. Узнай свой Telegram ID у [@userinfobot](https://t.me/userinfobot)
5. Вставь свой ID в список супер-админов:
   ```python
   SUPER_ADMINS = [твой_id]
   ```

## ▶️ Запуск

```bash
python bot.py
```

## 📋 Команды

### 👥 Тегалка
| Команда | Описание |
|---------|----------|
| `/all [текст]` | Тегнуть всех участников (по 5 в сообщении) |
| `/admins [текст]` | Тегнуть всех администраторов |
| `/tag @user [текст]` | Тегнуть конкретного пользователя |

### 📢 Рассылка
| Команда | Описание |
|---------|----------|
| `/broadcast [текст]` | Рассылка во все группы (только супер-адм) |
| `/schedule [мин] [текст]` | Отложенное сообщение |

### 👮 Модерация (только для админов)
| Команда | Описание |
|---------|----------|
| `/warn [причина]` | Предупреждение (3 → автобан) |
| `/mute [минуты]` | Заглушить пользователя |
| `/unmute` | Разглушить |
| `/kick` | Кикнуть (реплай) |
| `/ban [причина]` | Забанить навсегда |
| `/poll Вопрос? Вар1 Вар2` | Создать голосование |

### 🛠️ Настройки
| Команда | Описание |
|---------|----------|
| `/setwelcome [текст]` | Приветствие (`{name}` = имя) |
| `/admin` | Панель управления (кнопки) |
| `/stats` | Статистика |

---

## 🔧 Как работает тегалка

Бот **запоминает** участников автоматически когда они пишут в чат.  
При команде `/all` — тегает по **5 человек в одном сообщении** (настраивается в `/admin → Настройки`).

## ➕ Добавление в группу

1. Добавь бота в группу
2. Дай права **администратора**
3. Пиши команды!
